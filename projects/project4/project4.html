What is PyroDetect?

PyroDetect is an advanced project dedicated to fire detection through the application of cutting-edge deep learning technology. Designed as a comprehensive solution, PyroDetect features a Convolutional Neural Network (CNN) model crafted using TensorFlow and Keras. Tailored specifically for analyzing CCTV footage, this innovative system excels in real-time identification of potential fire incidents.
